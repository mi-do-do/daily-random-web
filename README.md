# 일상 랜덤 WebApp

Unity 앱과 분리된 모바일 우선 정적 웹앱입니다. 게임 계산은 접속자의 브라우저에서 실행되며 별도의 상시 서버가 필요하지 않습니다.

## 로컬 실행

정적 파일 서버로 `WebApp` 폴더를 열면 됩니다. HTTPS 또는 localhost에서 실행할 때 PWA 캐시가 활성화됩니다.

## 배포 구조

- Cloudflare Pages: `WebApp` 정적 파일
- Cloudflare R2: 트래픽이 늘어난 뒤 영상과 대용량 이미지 분리
- AdSense: `index.html` 승인 코드와 광고 슬롯 연동

현재 광고 영역은 정책 검토를 위한 안전한 하단 자리표시자입니다. 실제 AdSense 코드는 사이트와 도메인을 준비하고 승인 신청 시 연결합니다.
